# How to use

1. Install UiPath Community Edition https://www.uipath.com/developers/community-edition-download 
2. Ensure that you provide all required details and are able to open UiPath
3. If you wish to change your credentials, run Reset Credentials.bat
4. Run the file "Login to Pfizer Amazon VDS.vbs".Create a Desktop shortcut for convenience
5. This isn't a background automation.So please allow the bot to finish its job while you grab a tea!
